# Package Demo

This demo uses the `dom-physics` package installed from npm.

## Running the Demo

After installing the package:

```bash
npm install dom-physics
cd node_modules/dom-physics
npm run demo:package
```

Or if you're in the package directory:

```bash
npm run demo:package
```

Then open http://localhost:3000 in your browser.

## What This Demonstrates

- Basic World and Body creation
- Physics simulation with gravity, friction, and restitution
- Mouse interaction applying forces to bodies
- DOM structure preservation
- Transform-only manipulation
